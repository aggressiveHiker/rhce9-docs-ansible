Sensu.Sensu_Go

Collection version 1.13.1

  • Description

  • Plugin Index

  • Role Index

Description

Roles and modules for installing and using Sensu Go

Authors:

  • Paul Arthur <paul.arthur@flowerysong.com> (@flowerysong)

  • XLAB Steampunk <steampunk@xlab.si>

Issue Tracker Repository (Sources)

Plugin Index

These are the plugins in the sensu.sensu_go collection:

Modules

  • ad_auth_provider module – Manage Sensu AD authentication provider

  • asset module – Manage Sensu assets

  • asset_info module – List Sensu assets

  • auth_provider_info module – List Sensu authentication providers

  • bonsai_asset module – Add Sensu assets from Bonsai

  • check module – Manage Sensu checks

  • check_info module – List Sensu checks

  • cluster module – Manage Sensu Go clusters

  • cluster_info module – List available Sensu Go clusters

  • cluster_role module – Manage Sensu cluster roles

  • cluster_role_binding module – Manage Sensu cluster role bindings

  • cluster_role_binding_info module – List Sensu cluster role bindings

  • cluster_role_info module – List Sensu cluster roles

  • datastore module – Manage Sensu external datastore providers

  • datastore_info module – List external Sensu datastore providers

  • entity module – Manage Sensu entities

  • entity_info module – List Sensu entities

  • etcd_replicator module – Manage Sensu Go etcd replicators

  • etcd_replicator_info module – List available Sensu Go etcd replicators

  • event module – Manage Sensu events

  • event_info module – List Sensu events

  • filter module – Manage Sensu filters

  • filter_info module – List Sensu info

  • handler_info module – List Sensu handlers

  • handler_set module – Manage Sensu handler set

  • hook module – Manage Sensu hooks

  • hook_info module – List Sensu hooks

  • ldap_auth_provider module – Manage Sensu LDAP authentication provider

  • mutator module – Manage Sensu mutators

  • mutator_info module – List Sensu mutators

  • namespace module – Manage Sensu namespaces

  • namespace_info module – List Sensu namespaces

  • oidc_auth_provider module – Manage Sensu OIDC authentication provider

  • pipe_handler module – Manage Sensu pipe handler

  • role module – Manage Sensu roles

  • role_binding module – Manage Sensu role bindings

  • role_binding_info module – List Sensu role bindings

  • role_info module – List Sensu roles

  • secret module – Manage Sensu Go secrets

  • secret_info module – List available Sensu Go secrets

  • secrets_provider_env module – Manage Sensu Env secrets provider

  • secrets_provider_info module – List Sensu secrets providers

  • secrets_provider_vault module – Manage Sensu VaultProvider secrets providers

  • silence module – Manage Sensu silences

  • silence_info module – List Sensu silence entries

  • socket_handler module – Manage Sensu TCP/UDP handler

  • tessen module – Manage Sensu’s Tessen configuration

  • user module – Manage Sensu users

  • user_info module – List Sensu users

Role Index

These are the roles in the sensu.sensu_go collection:

  • agent role – Install, configure, and start Sensu Go agent

  • backend role – Install, configure, and start Sensu Go backend

  • install role – Enable Sensu Go repos and install selected packages
