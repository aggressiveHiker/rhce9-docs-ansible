Cisco.Ucs

Collection version 1.8.0

  • Description

  • Plugin Index

Description

modules for Cisco UCS Manager

Authors:

  • David Soper (@dsoper2)

  • John McDonough (@movinalot)

Issue Tracker Repository (Sources)

Plugin Index

These are the plugins in the cisco.ucs collection:

Modules

  • ucs_disk_group_policy module – Configures disk group policies on Cisco UCS Manager

  • ucs_dns_server module – Configure DNS servers on Cisco UCS Manager

  • ucs_graphics_card_policy module – Manages UCS Graphics Card Policies on UCS Manager

  • ucs_ip_pool module – Configures IP address pools on Cisco UCS Manager

  • ucs_lan_connectivity module – Configures LAN Connectivity Policies on Cisco UCS Manager

  • ucs_mac_pool module – Configures MAC address pools on Cisco UCS Manager

  • ucs_managed_objects module – Configures Managed Objects on Cisco UCS Manager

  • ucs_ntp_server module – Configures NTP server on Cisco UCS Manager

  • ucs_org module – Manages UCS Organizations for UCS Manager

  • ucs_query module – Queries UCS Manager objects by class or distinguished name

  • ucs_san_connectivity module – Configures SAN Connectivity Policies on Cisco UCS Manager

  • ucs_scrub_policy module – Manages UCS Scrub Policies on UCS Manager

  • ucs_serial_over_lan_policy module – Manages UCS Serial Over Lan Policies on UCS Manager

  • ucs_server_maintenance module – Creates Server Maintenance Policy on Cisco UCS Manager

  • ucs_service_profile_association module – Configures Service Profile Association on Cisco UCS Manager

  • ucs_service_profile_from_template module – Configures Service Profiles from templates on Cisco UCS Manager

  • ucs_service_profile_template module – Configures Service Profile Templates on Cisco UCS Manager

  • ucs_sp_vnic_order module – Configures vNIC order for service profiles and templates on Cisco UCS Manager

  • ucs_storage_profile module – Configures storage profiles on Cisco UCS Manager

  • ucs_system_qos module – Configures system QoS settings

  • ucs_timezone module – Configures timezone on Cisco UCS Manager

  • ucs_uuid_pool module – Configures server UUID pools on Cisco UCS Manager

  • ucs_vhba_template module – Configures vHBA templates on Cisco UCS Manager

  • ucs_vlan_find module – Find VLANs on Cisco UCS Manager

  • ucs_vlan_to_group module – Add VLANs to a VLAN Group. Requires VLAN and VLAN Group to already be created on UCS prior to running module.

  • ucs_vlans module – Configures VLANs on Cisco UCS Manager

  • ucs_vnic_template module – Configures vNIC templates on Cisco UCS Manager

  • ucs_vsans module – Configures VSANs on Cisco UCS Manager

  • ucs_wwn_pool module – Configures WWNN or WWPN pools on Cisco UCS Manager
