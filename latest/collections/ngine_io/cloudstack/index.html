Plugin Index

These are the plugins in the ngine_io.cloudstack collection:

Modules

  • cs_account module – Manages accounts on Apache CloudStack based clouds.

  • cs_affinitygroup module – Manages affinity groups on Apache CloudStack based clouds.

  • cs_cluster module – Manages host clusters on Apache CloudStack based clouds.

  • cs_configuration module – Manages configuration on Apache CloudStack based clouds.

  • cs_disk_offering module – Manages disk offerings on Apache CloudStack based clouds.

  • cs_domain module – Manages domains on Apache CloudStack based clouds.

  • cs_facts module – Gather facts on instances of Apache CloudStack based clouds.

  • cs_firewall module – Manages firewall rules on Apache CloudStack based clouds.

  • cs_host module – Manages hosts on Apache CloudStack based clouds.

  • cs_image_store module – Manages CloudStack Image Stores.

  • cs_instance module – Manages instances and virtual machines on Apache CloudStack based clouds.

  • cs_instance_info module – Gathering information from the API of instances from Apache CloudStack based clouds.

  • cs_instance_nic module – Manages NICs of an instance on Apache CloudStack based clouds.

  • cs_instance_nic_secondaryip module – Manages secondary IPs of an instance on Apache CloudStack based clouds.

  • cs_instance_password_reset module – Allows resetting VM the default passwords on Apache CloudStack based clouds.

  • cs_instancegroup module – Manages instance groups on Apache CloudStack based clouds.

  • cs_ip_address module – Manages public IP address associations on Apache CloudStack based clouds.

  • cs_iso module – Manages ISO images on Apache CloudStack based clouds.

  • cs_loadbalancer_rule module – Manages load balancer rules on Apache CloudStack based clouds.

  • cs_loadbalancer_rule_member module – Manages load balancer rule members on Apache CloudStack based clouds.

  • cs_network module – Manages networks on Apache CloudStack based clouds.

  • cs_network_acl module – Manages network access control lists (ACL) on Apache CloudStack based clouds.

  • cs_network_acl_rule module – Manages network access control list (ACL) rules on Apache CloudStack based clouds.

  • cs_network_offering module – Manages network offerings on Apache CloudStack based clouds.

  • cs_physical_network module – Manages physical networks on Apache CloudStack based clouds.

  • cs_pod module – Manages pods on Apache CloudStack based clouds.

  • cs_portforward module – Manages port forwarding rules on Apache CloudStack based clouds.

  • cs_project module – Manages projects on Apache CloudStack based clouds.

  • cs_region module – Manages regions on Apache CloudStack based clouds.

  • cs_resourcelimit module – Manages resource limits on Apache CloudStack based clouds.

  • cs_role module – Manages user roles on Apache CloudStack based clouds.

  • cs_role_permission module – Manages role permissions on Apache CloudStack based clouds.

  • cs_router module – Manages routers on Apache CloudStack based clouds.

  • cs_securitygroup module – Manages security groups on Apache CloudStack based clouds.

  • cs_securitygroup_rule module – Manages security group rules on Apache CloudStack based clouds.

  • cs_service_offering module – Manages service offerings on Apache CloudStack based clouds.

  • cs_snapshot_policy module – Manages volume snapshot policies on Apache CloudStack based clouds.

  • cs_sshkeypair module – Manages SSH keys on Apache CloudStack based clouds.

  • cs_staticnat module – Manages static NATs on Apache CloudStack based clouds.

  • cs_storage_pool module – Manages Primary Storage Pools on Apache CloudStack based clouds.

  • cs_template module – Manages templates on Apache CloudStack based clouds.

  • cs_traffic_type module – Manages traffic types on CloudStack Physical Networks

  • cs_user module – Manages users on Apache CloudStack based clouds.

  • cs_vlan_ip_range module – Manages VLAN IP ranges on Apache CloudStack based clouds.

  • cs_vmsnapshot module – Manages VM snapshots on Apache CloudStack based clouds.

  • cs_volume module – Manages volumes on Apache CloudStack based clouds.

  • cs_vpc module – Manages VPCs on Apache CloudStack based clouds.

  • cs_vpc_offering module – Manages vpc offerings on Apache CloudStack based clouds.

  • cs_vpn_connection module – Manages site-to-site VPN connections on Apache CloudStack based clouds.

  • cs_vpn_customer_gateway module – Manages site-to-site VPN customer gateway configurations on Apache CloudStack based clouds.

  • cs_vpn_gateway module – Manages site-to-site VPN gateways on Apache CloudStack based clouds.

  • cs_zone module – Manages zones on Apache CloudStack based clouds.

  • cs_zone_info module – Gathering information about zones from Apache CloudStack based clouds.
