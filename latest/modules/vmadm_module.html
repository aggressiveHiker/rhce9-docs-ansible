vmadm – Manage SmartOS virtual machines and zones¶

  • Synopsis
  • Requirements
  • Parameters
  • Examples
  • Return Values
  • Status

Synopsis¶

  • Manage SmartOS virtual machines through vmadm(1M).

Requirements¶

The below requirements are needed on the host that executes this module.

  • python >= 2.6

Examples¶

- name: create SmartOS zone
  vmadm:
    brand: joyent
    state: present
    alias: fw_zone
    image_uuid: 95f265b8-96b2-11e6-9597-972f3af4b6d5
    firewall_enabled: yes
    indestructible_zoneroot: yes
    nics:
      - nic_tag: admin
        ip: dhcp
        primary: true
    internal_metadata:
      root_pw: 'secret'
    quota: 1

- name: Delete a zone
  vmadm:
    alias: test_zone
    state: deleted

- name: Stop all zones
  vmadm:
    uuid: '*'
    state: stopped
